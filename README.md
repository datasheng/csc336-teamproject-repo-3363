CSC336 Team Project – Restaurant Ordering & Delivery System

This repository contains the frontend of our CSC336 team project (Team 3363).
It was built using React and Vite, and connects to a backend developed by other team members.

HOW TO DOWNLOAD THE PROJECT

Go to the repository link:
https://github.com/datasheng/csc336-teamproject-repo-3363

Click the green “Code” button.
You can choose one of the following:

Download ZIP → extract it to your computer.

OR Clone with Git (recommended for developers):
git clone https://github.com/datasheng/csc336-teamproject-repo-3363.git

HOW TO RUN THE FRONTEND LOCALLY

Open the project folder in your terminal.
Example:
cd frontend

Install dependencies (run this only once):
npm install

Start the development server:
npm run dev

Open your browser and go to:
http://localhost:5173/

If everything works correctly, you should see the restaurant ordering website running locally.

HOW TO UPLOAD OR UPDATE YOUR CODE ON GITHUB

(Only team members with GitHub access can push updates.)

Make sure you are in the project directory.
Example:
cd C:\Users<your_username>\Downloads\restaurant-basic-docker\restaurant-basic-docker

Check the current Git status:
git status

Stage your changes:
git add .

Commit with a short message:
git commit -m "Update frontend code and fix styling"

Push the changes to GitHub:
git push

That’s it! Your changes will appear in the shared GitHub repository.
